# C# Order System

This project contains three folders, with each folder containing classes and logic:
- OrderConsole
- OrderProduct
- OrderSales

## How to run
Step 1. Open a terminal and clone the repository with the following command:
```bash
  git clone https://github.com/w-oby/CSharp-assignment
```

Step 2. Head to the root of the project `CSharp-assignment/`

Step 3. Run the project by executing the following commands:
```bash
  dotnet build
  dotnet run --project OrderConsole
```
